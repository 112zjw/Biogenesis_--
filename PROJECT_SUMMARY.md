# Biogenesis 项目总结

## 项目概述
Biogenesis 是一款基于生物信息学的进化模拟游戏，玩家扮演"进化工程师"，通过编辑物种的DNA序列帮助它们适应不断变化的环境。

## 实现的功能

### 核心系统
1. **DNA系统** (`DNA`类)
   - 随机生成DNA序列 (ATCG碱基)
   - DNA突变功能
   - GC含量计算
   - 支持自定义序列

2. **生物系统** (`Organism`类)
   - 物种个体管理
   - 适应度计算（基于GC含量和温度）
   - 存活/死亡判定
   - 年龄追踪
   - 自然突变机制

3. **环境系统** (`Environment`类)
   - 动态环境参数（理想GC含量、温度）
   - 代际环境变化
   - 多样化的生存挑战

4. **游戏系统** (`Game`类)
   - 完整的游戏循环
   - 互动式命令行界面
   - DNA编辑功能（每代3次机会）
   - 进度追踪和状态显示

### 游戏机制
- **适应度计算**: 基于DNA的GC含量与环境理想值的差异
- **存活判定**: 适应度低于30有50%死亡概率
- **年龄系统**: 年龄超过10代有30%死亡概率
- **突变机制**: 10%概率发生自然突变
