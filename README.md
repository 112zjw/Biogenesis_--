# Biogenesis_--
Biogenesis是一款结合了生物学概念与策略要素的进化模拟游戏。在这个游戏中，你将扮演一名"进化工程师"，通过编辑物种的 DNA 序列，帮助它们适应不断变化且充满挑战的生存环境。

## 游戏特性

- **DNA序列编辑**: 直接修改生物的DNA序列，改变其GC含量以适应环境
- **环境适应**: 环境条件会随机变化，物种需要不断进化以适应新环境
- **适应度系统**: 物种的生存取决于其DNA与环境的匹配程度
- **代际演化**: 物种会随着时间推移老化，还可能发生自然突变

## 游戏机制

### DNA系统
- DNA由四种碱基组成: **A**(腺嘌呤), **T**(胸腺嘧啶), **C**(胞嘧啶), **G**(鸟嘌呤)
- GC含量 = (G数量 + C数量) / DNA总长度
- 不同环境有不同的理想GC含量

### 适应度计算
- 物种的GC含量越接近环境理想值，适应度越高
- 温度也会影响物种的适应度
- 适应度低于30的物种有50%概率死亡

### 环境系统
- 环境的理想GC含量会在每代变化
- 温度也会随机波动
- 物种需要不断调整以适应新环境

## 安装与运行

### 环境要求
- Python 3.6 或更高版本

### 运行游戏

```bash
python3 biogenesis.py
```

或者使用可执行权限:

```bash
chmod +x biogenesis.py
./biogenesis.py
```

## 游戏玩法

1. **游戏开始**: 你将获得3个初始物种，每个都有随机的DNA序列
2. **查看状态**: 观察当前环境条件和物种的适应度
3. **编辑DNA**: 每代你有3次机会修改物种的DNA碱基
   - 选择要编辑的物种
   - 选择要修改的位置
   - 输入新的碱基(A/T/C/G)
4. **进入下一代**: 
   - 环境条件会变化
   - 物种会根据适应度判断是否存活
   - 可能发生自然突变
5. **游戏目标**: 尽可能让物种存活更多代数

## 游戏提示

- **GC含量策略**: 观察环境的理想GC含量，调整物种DNA使其GC含量接近理想值
  - 增加GC含量: 将A或T改为G或C
  - 降低GC含量: 将G或C改为A或T
- **保持多样性**: 不要让所有物种的DNA完全相同，多样性有助于应对环境变化
- **提前规划**: 观察环境变化趋势，提前调整DNA
